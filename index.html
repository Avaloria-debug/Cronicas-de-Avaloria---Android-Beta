Clave del Narrador

Para despertar al Narrador necesita una clave de Groq. Hágalo así: entre a console.groq.com/keys, cree su cuenta, genere una API key y péguela acá.

La clave se guarda solo en este navegador, en localStorage. El juego no tiene servidor propio.
